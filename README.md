# Trend Spam Radar

A tiny viral tool that turns suspicious trending-repo dumps into a visual ‘trend spam radar,’ helping developers spot star-farmed malware-bait and clone clusters before they click download links.

## Why This Repo

This project is generated by a trend-driven pipeline. It targets **Developers, security researchers, GitHub trend watchers, open-source maintainers, and tech newsletter authors** and focuses on the niche: **Open-source security / GitHub trend integrity / repository reputation analysis**.

## Key Features

- Paste a GitHub trending JSON/export and get an instant suspicious-cluster report: repeated descriptions, zero-fork star spikes, same-language template farms, keyword stuffing, and near-duplicate repo names
- CLI plus single-page static web UI that generates a shareable Markdown badge/report for README, issues, or social posts
- Similarity scoring using lightweight MinHash/TF-IDF heuristics with no API key required; optional GitHub token mode to enrich age, releases, commits, and owner metadata
- Built-in rules for common scammy patterns such as game trainer/cheat lures, “undetected” claims, fake download repos, and mass-created variants
- Exports SARIF/JSON/Markdown so maintainers, security researchers, and newsletter authors can reuse results in workflows
